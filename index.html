Empieza fácilmente con esta guía

Configurando git

git config

Se utiliza Para ver todas las opciones de git

git config --global user.name git config --global user.email

Agregar usuario y correo electronico

git config --list

Ver configuracion actual

git config --global core.editor "code.wait"

Agregar editor de código

Ver repositorio que vamos a trabajar

"dir"

Visualizar las carpetas que tenemos

cd c:

Ir a la carpeta principal

cd "Nombre de la carpeta"

Ingresamos a la carpeta asignada

pdw

Crea un repositorio nuevo

git init

Hacer checkout a un repositorio

$ git clone "Link de clone"

Clona la carpeta que tengo en la nube como repositorio. Copio la dirección o ubicación, doy enter y descarga en mi carpeta de repositorio local

Crea el archivo dentro de la carpeta que descargamos

$ code index.html

Para abrir el editor de código que dejamos por defecto
